To connect to a remote host, do the following steps:
  1. Select the connection type (telnet, ssh, or rlogin)
  2. Enter the server name (e.g.: login.myworkplace.com)
  3. Enter the port number you wish to connect (only necessary if you're trying to connect to a different port number than the default)
  4. Click "Connect" or hit Enter in the host name entry box.
  5. If you use SSH connection, you will be prompted for a user name. Enter the user name and click "OK" button. If you don't want to be prompted for username, just say "your_user_name@some_ssh_host.com" and click Enter.
  6. When you're done, you can click "Disconnect"
